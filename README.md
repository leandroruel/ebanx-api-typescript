## MINI API

## Installation

install all dependencies:

```bash
$ npm install
```

then run migrations for sqlite:

```bash
$ npm run prisma generate
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
